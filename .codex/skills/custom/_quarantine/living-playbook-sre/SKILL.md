# Living Playbook SRE

## Purpose
Maintain incident-response runbooks as living operational assets.

## Use This Skill When
- Standardizing response actions for recurring incidents.
- Updating runbooks from incident retrospectives.

## Workflow
1. Classify incident type and blast radius.
2. Map to runbook actions, owners, and rollback controls.
3. Execute and log action outcomes.
4. Update playbook with lessons and trigger refinements.

## Scope Boundary

**In scope**
- Runbook mapping and incident playbook maintenance.

**Out of scope**
- Deep causal analysis (use `sre-causal-rca`).
