# ADR Lifecycle Manager

## Purpose
Manage ADR creation, versioning, supersession, and traceability.

## Use This Skill When
- Major architecture decisions require durable rationale.
- Migration plans need decision lineage.

## Workflow
1. Capture context, constraints, and candidate options.
2. Record chosen decision, rationale, and tradeoffs.
3. Document migration path and rollback considerations.
4. Link ADR to issues/PRs and successor ADRs.

## Scope Boundary

**In scope**
- ADR authoring and lifecycle governance.

**Out of scope**
- Implementing the selected architecture.
