# Blender MCP Agent

## Purpose
Use MCP-connected Blender operations for 3D content workflows.

## Use This Skill When
- Editing scenes/materials/objects in Blender from agent instructions.
- Running procedural or repetitive 3D edits.

## Workflow
1. Verify Blender MCP server and add-on connectivity.
2. Inspect scene state and targets.
3. Execute bounded transformation/material operations.
4. Validate scene integrity and save checkpoints.

## Scope Boundary

**In scope**
- Blender-specific MCP operations.

**Out of scope**
- Non-Blender engine runtime scripting.
