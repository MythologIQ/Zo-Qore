# ElevenLabs Evaluator

## Purpose
Measure voice quality and performance against production KPIs.

## Use This Skill When
- Verifying voice-agent quality before release.
- Tracking emotional prosody and latency compliance.

## Workflow
1. Define KPI targets and evaluation rubric.
2. Run sample-set evaluations across scenarios.
3. Measure latency stages and compare to budget.
4. Produce pass/fail report and remediation priorities.

## Scope Boundary

**In scope**
- Voice quality/performance evaluation and KPI reporting.

**Out of scope**
- Initial voice asset generation or API credential setup.
